# Portfolio

Hi! My name is Jason, and this is my portfolio

---

## Tech Stack

- Frontend: React + TypeScript + Vite
- Styling: Tailwind CSS

---

## Setting up & Installation

- Prereqs: Node.js and npm

1. clone the repository
2. cd into project folder
3. run ```npm install```
4. run ```npm run dev```
